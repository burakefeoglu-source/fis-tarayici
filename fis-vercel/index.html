Durum

Bu versiyon Netlify Functions kullanıyor.
API anahtarını Netlify'da güvenli saklayacaksın — tarayıcıya hiç gelmiyor.

Aşağıdaki adımları tamamla:

Adım 1 — Netlify'da env variable ekle

1. app.netlify.com → Siten → Site configuration
2. Sol menü: Environment variables
3. Add a variable tıkla
4. Key: ANTHROPIC_API_KEY
5. Value: API anahtarın (sk-ant-...)
6. Save → sonra Deploys → Trigger deploy

Adım 2 — Siteyi yeniden deploy et

Bu ZIP'i Netlify'a yüklerken fis-v4 klasörünün içindeki dosyaları sürükle (klasörü değil, içini).
